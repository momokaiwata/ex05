# ゲームのタイトル
    「RPG of くそげー」
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
    RPG内の戦闘シーン
    私が担当している実装を説明すると、
    slimeから攻撃されている状態で、
    5秒ごとにHPが3ずつ減っていく。
    しかし、画面上の防御を押すことにより
    HPが減らないようにしてくれます。
    また、他の攻撃や回復を押すことで
    防御されなくなります。

##　ゲームの実装
### 共通機能
* 初期値
* buttun関数
* sction関数

### 担当追加機能
* 攻撃　→　サクライ
* 防御　→　イワタ
* 死ぬ　→　コヤマ
* 回復　→　ママタ
* テイム　→　井原
